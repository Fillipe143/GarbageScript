# GarbageScript
Garbage script is a programming language made with the idea of ​​being as bad as possible to program, but maintaining usefulness

## Examples
Hello world:
```
i"standard.gb"

x"Hello world!\n" %std_write#
```

Read file:
```
i"standard.gb"
i"iostream.gb"

x"example.txt" %io_read_file#
%std_write#
```
